# Gemini API Python Project

This is a sample Python project that demonstrates how to call the Google Gemini API.

## Setup

1. Install dependencies:
   ```
   pip install -r requirements.txt
   ```

2. Get a Gemini API key from [Google AI Studio](https://makersuite.google.com/app/apikey).

3. Replace 'YOUR_API_KEY' in `main.py` with your actual API key.

## Usage

Run the script:
```
python main.py
```

This will generate a short story using the Gemini API.
